\# Sistema de Reconocimiento Facial para Control de Ingreso Escolar



\## Descripción

Sistema que utiliza reconocimiento facial con ArcFace para controlar el ingreso de alumnos a la escuela. Registra la entrada con fecha y hora, y en el futuro podrá activar una cerradura eléctrica mediante Arduino.



\## Tecnologías utilizadas

\- Python 3.12

\- DeepFace (ArcFace)

\- OpenCV (cámara web)

\- MySQL (pendiente)

\- Arduino (pendiente)
